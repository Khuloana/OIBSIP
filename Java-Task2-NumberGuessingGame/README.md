# Number Guessing Game

## Project Description

This project is a Java console application where the computer randomly generates a number, and the player attempts to guess it.

The game provides hints indicating whether the guessed number is too high or too low until the correct number is found.

## Features

- Random Number Generation
- User Input
- Guess Validation
- Hint Messages
- Attempt Counter
- Play Again Option

## Technologies Used

- Java
- NetBeans IDE
- Object-Oriented Programming (OOP)

## How to Run

1. Open the project in NetBeans.
2. Run the Main.java file.
3. Enter your guesses until you find the correct number.
