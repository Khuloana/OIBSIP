# ATM Interface

## Project Description

This project is a console-based ATM Interface developed in Java using Object-Oriented Programming (OOP).

The system allows users to log in using a User ID and PIN and perform common banking operations.

## Features

- User Login
- Deposit Money
- Withdraw Money
- Transfer Money
- View Transaction History
- Check Account Balance
- Exit the Application

## Technologies Used

- Java
- NetBeans IDE
- Object-Oriented Programming (OOP)
- ArrayList

## How to Run

1. Open the project in NetBeans.
2. Run the Main.java file.
3. Enter the User ID and PIN.
4. Choose an option from the ATM menu.
